---

## 核心判断

真正的AI native应用现在还在婴儿期，大概率还没上市。

现在二级市场能买到的AI应用层，只有两种：

- 旧公司贴了AI皮（Figma就是教材）
- 极少数稀缺标的享受稀缺性溢价（PLTR）

所以现在最贵的是PLTR，但它贵的是稀缺性，不是基本面。

---

## Figma是活教材

从140跌到25，这就是"旧公司加AI皮"的定价修正过程：

```
旧公司贴AI标签 → 短期炒作
→ 市场发现没有真正的AI护城河
→ 回归基本面
```

_Figma的护城河是用户习惯和协作网络，不是AI能力本身。Cursor出来之后，"AI加持的创作工具"这个故事就开始被市场怀疑了。_

---

## 筛选真AI Native公司的维度

**创始团队结构**

真正的AI native公司大概率是年轻人和老登联合创业的：

- 年轻人：技术判断力，真正理解新范式
- 老登：资源、人脉、行业经验

纯老登团队大概率是套皮，脑子跟不上时代。 纯年轻人团队可能缺少落地能力。

_这个组合在历史上出现过很多次，PC时代、互联网时代都有类似规律。_

**护城河检验**

对着矿炼金厂模型问三个问题：

1. 你的矿是什么？（私有数据/场景）
2. 为什么竞争对手拿不到？
3. 你的炼厂运营成本谁在付？

答不清楚的，AI皮的概率很高。

## 信号时间轴：不要等Gartner

Gartner是给企业CIO看的采购指南，等它说"爬升期"已经太晚：

```
你朋友圈的技术人在讨论    → 最早（现在）
技术在GitHub开始爆发      → 很早
顶级VC的portfolio变化     → 早
VC开始集中押注            → 早一步
Gartner说爬升期           → 太晚
公司IPO                   → 更晚
散户开始讨论              → 最晚
```

_Gartner单份报告几千美元，对你的用途来说信号还滞后，不值得订阅。_

**免费的上游替代**

```
a16z年度AI报告
Sequoia年度报告
FirstMark MAD Landscape（每年10月）
CB Insights免费newsletter
顶级VC的Twitter/X动态
```

加起来比Gartner更早、更准、还不要钱。

## 现在的正确姿势

真正的AI native应用还没上市，等它们上市时可能已经是佩雷斯整合期了。

**所以现在要做的不是买股票，是提前建仓认知：**

```
研究清楚婴儿期公司的产业逻辑
→ 等它们上市时已经有判断
→ 不会在发行价冲进去被套
```

_这正是搭信息系统、逆向工程研究报告的真实原因——提前建仓认知，不是提前建仓股票。_
