# 🌱 我的狗窝

金窝银窝不如自己的狗窝。

这里是我的个人知识库。不追求完整，不写给别人看——一旦有那种念头我一个字也写不出来，因为每个人的base不一样，要整理成wiki收益太低。

所以我就写给自己，充满偏见

---

笔记的构建方式借鉴了Linux包管理的理念：

`apt install 付鹏长白山发言 -y`

会自动索引出所有依赖：康波周期、佩雷斯技术革命、熊彼特创造性破坏……每一个论点都有理论根基可以追溯。只有论点没有论据的文章站不住脚，但直接把文章丢给LLM步子太大容易产生幻觉。
